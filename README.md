## Projects

### OIDC Authentication Lab
Configured an application using Auth0 OIDC.
Tested authorization flow and analyzed ID/access tokens.

### SAML SSO Federation Lab
Configured a SAML trust relationship between Auth0 (IdP) and a Service Provider.
Performed metadata exchange, tested SSO, and inspected SAML assertions and claims.

### OAuth 2.0 Client Credentials Lab
Configured an API and Machine-to-Machine application in Auth0.
Requested and analyzed OAuth access tokens, scopes, and audience claims using the Client Credentials flow.
